# Real-Time Streaming Data Pipeline

## Overview
This project simulates a real-time data streaming pipeline where data is continuously ingested, processed, and analyzed.

## Features
- Simulated real-time data streaming
- Event-based data processing
- Scalable architecture design
- Streaming analytics logic

## Tech Stack
- Python
- Pandas
- Time module (stream simulation)

## Dataset
- Student Performance Dataset (Kaggle)

## Workflow
1. Stream data row-by-row
2. Process incoming records
3. Perform real-time analytics
4. Output processed results

## Results
- Demonstrates handling of streaming data
- Enables real-time analytics simulation

## How to Run
1. Upload dataset in Colab
2. Run streaming script
3. Observe real-time processing

## Future Improvements
- Integrate Apache Kafka
- Use Spark Streaming
